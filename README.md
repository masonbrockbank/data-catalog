<img width="520" alt="image" src="https://github.com/masonbrockbank/data-catalog/assets/90272781/534af092-0d36-402b-b009-54cfec310142"># Follow these steps to run this project locally

1. Go to this link: https://developer.domo.com/portal/d9520f5752d56-get-access-token
2. Fill out the username and password in the top right: username being a domo client id and password being a domo client secret
3. In the Request sample shell command directly below: copy the authorization header (just the hash value after basic)
4. Paste this value in step 3 in the env file
